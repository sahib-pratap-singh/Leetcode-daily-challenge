<h1>:heavy_check_mark: 338. Counting Bits</h1>
<blockquote>Given an integer n, return an array ans of length n + 1 such that for each i (0 <= i <= n), ans[i] is the number of 1's in the binary representation of i.
</blockquote><br>

* **Example 1**:<br>

        Input: n = 2
        Output: [0,1,1]
        Explanation:
        0 --> 0
        1 --> 1
        2 --> 10
      
* **Example 2**:<br>

        Input: n = 5
        Output: [0,1,1,2,1,2]
        Explanation:
        0 --> 0
        1 --> 1
        2 --> 10
        3 --> 11
        4 --> 100
        5 --> 101

### **Constraints**

        0 <= n <= 105
